Find Max Bubble Sort

For loop exercise: Find Max

In this exercise, we try to find the maximum number in a list.

Method

First set the first element as the variable MAX (i.e. MAX = list[0]), then we will compare the following elements one by one. If the compared element is larger than the value of the variable MAX, the value of the variable MAX is then changed to the current compared element.
